== README

*Aplicação Gateway  - desenvolvido como exemplo para Trabalho de Conclusão de Curso.

*Fornece respostas do sistema de Venda de Produtos, utilizando os Microsserviços via API REST (JSON).

*Ruby version: ruby 2.1.1p76 

*Rails version: Rails 4.2.6

*System dependencies: gem http, SQLite3.

*Configuration: rake bundle install

*Database creation: rake db:migrate

*Database initialization: rake db:seed
